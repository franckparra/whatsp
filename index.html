Perguntas Frequentes

O AI Personas é gratuito?

Sim, a interface e o acesso às personas são gratuitos. Você precisa apenas de uma chave API (API Key) de um provedor compatível (OpenRouter, Inception Labs ou Google AI) para que a IA processe as conversas. Alguns provedores oferecem créditos gratuitos iniciais (ex: Inception Labs oferece 10 milhões de tokens grátis).

Minha chave API é segura? Vocês armazenam ela?

Sim, é totalmente segura. Sua chave API é armazenada exclusivamente no localStorage do seu navegador (dispositivo local). Ela nunca é enviada para nossos servidores nem para terceiros, sendo usada apenas para fazer requisições diretas do seu navegador para a API do provedor escolhido.

Quais provedores de IA são suportados?

Suportamos três provedores principais: 1) Open Router (acesso a centenas de modelos: GPT-4, Claude 3, Llama 3, Mistral); 2) Inception Labs (modelos otimizados para conversação natural, com 10M tokens grátis); 3) Google AI (modelos Gemini Pro). Você escolhe qual usar nas configurações do app.

O que são "Personas Sintéticas" e como diferem de chatbots comuns?

Nossas personas não são prompts genéricos. Cada uma possui um Perfil Sintético estruturado: nome, idade, cidade, história de vida, traços de personalidade baseados no modelo Big Five (Abertura, Conscienciosidade, Extroversão, Amabilidade, Neuroticismo), valores, medos, estilo de fala e memórias simuladas. Isso garante consistência comportamental de longo prazo, diferentemente de chatbots que "esquecem" a personalidade após poucas mensagens.

Preciso instalar algo? Funciona no celular?

Não requer instalação. É uma Progressive Web App (PWA) que roda direto no navegador (Chrome, Safari, Firefox, Edge) em desktop, tablet ou celular. Você pode "Instalar" o app pela opção do navegador para ter ícone na tela inicial e experiência fullscreen.
