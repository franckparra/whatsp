4. Arquitetura Privacy-by-Design: Client-Side Only (BYOK) LGPD Art. 6º, 46, 50

Princípio: "Seus Dados, Seu Dispositivo, Seu Controle"

O AI Personas é uma Single Page Application (SPA) estática hospedada no GitHub Pages atrás de proxy Cloudflare. Não possuímos backend, banco de dados, servidor de aplicação ou API própria.

Chaves API (Segredos)

  • Inseridas por você na tela de configurações.
  • Salvas no localStorage (criptografado pelo navegador no disco).
  • Lidas apenas pelo JavaScript rodando no seu navegador para montar o header Authorization: Bearer sk-... nas requisições diretas para openrouter.ai, inceptionlabs.ai, generativelanguage.googleapis.com.
  • Nunca enviadas para github.io, cloudflare.net ou qualquer domínio nosso.

Histórico de Conversas

  • Mantido em memória (estado React/JS) durante a sessão.
  • Opcionalmente salvo no localStorage se você ativar "Persistir Chat" (padrão: off).
  • Nós não temos acesso. Só você (e quem tiver acesso físico/remoto ao seu dispositivo/navegador).

Implicação Prática: Como não processamos seus dados em nossos servidores, não temos como "entregar" seu histórico de chat ou "apagar" suas chaves por você. O controle é totalmente seu: DevTools > Application > LocalStorage > Limpar ou botão "Limpar Dados" no App (se implementado).

5. Compartilhamento com Suboperadores e Transferência Internacional LGPD Art. 7º §1º, 16, 33

Para que o App funcione, seus dados (IP, Chaves API, Conteúdo do Chat) trafegam para terceiros essenciais. Não vendemos, alugamos ou compartilhamos dados para publicidade de terceiros.

Cloudflare, Inc. (Proxy Reverso, CDN, WAF, DNS)

Função: Todo tráfego HTTPS passa pela rede Cloudflare antes de chegar ao GitHub Pages. Proteção DDoS, WAF (Firewall de Aplicação), Cache, Certificado TLS.

Dados Recebidos: Endereço IP completo, Headers HTTP, Cookies de segurança (__cf_bm, cf_clearance), Geolocalização (IP-based), Impressão digital do navegador (Bot Management).

Localização / Transferência: EUA 🇺🇸 (Sede) + Edge Global. Cláusulas Contratuais Padrão (SCC) e Data Processing Addendum (DPA) vigentes. Certificação ISO 27001, SOC 2 Type II, PCI DSS.

Retenção: Logs de borda (Edge Logs) geralmente < 24h a 7 dias. Logs de segurança (WAF events) conforme plano.

7. Segurança da Informação e Retenção LGPD Art. 46, 48

Medidas Técnicas (Nossa Responsabilidade)

  • TLS 1.3 / HTTPS Forçado (HSTS): Via Cloudflare (Certificados Universal SSL / Custom).
  • Headers de Segurança: Content-Security-Policy (CSP) restritivo (via _headers Netlify/Cloudflare Workers ou meta tag), X-Frame-Options: DENY, Referrer-Policy: strict-origin-when-cross-origin, Permissions-Policy.
  • WAF Gerenciado (Cloudflare): Regras OWASP Top 10, Rate Limiting, Bot Fight Mode.
  • Supply Chain: Dependências via CDN (unpkg.com, fonts.googleapis.com) com integrity (SRI) onde aplicável. Subresource Integrity recomendado para produção.
  • Sem Backend = Superfície de Ataque Mínima: Sem SQL Injection, Sem RCE no servidor, Sem vazamento de DB.

Responsabilidades Compartilhadas (Sua Parte)

  • Guarde suas Chaves API: Trate como senhas. Não cole em Issues públicos, logs, prints.
  • Revogue Chaves Comprometidas: No painel do provedor (OpenRouter/Inception/Google) imediatamente se suspeitar de vazamento.
  • Limpeza de Dispositivo: Use "Limpar Dados do App" ou limpe localStorage ao trocar de dispositivo/compartilhar computador.
  • Navegador Atualizado: Mantenha Chrome/Firefox/Safari/Edge atualizados para patches de segurança (Spectre, XS-Leaks).
  • Dispositivo Seguro: Antivírus, Firewall, Disco Criptografado (BitLocker/FileVault/LUKS).
Retenção de Logs de Segurança (Nós/Cloudflare/GitHub): Conforme Tabela Seção 3. Não mantemos dossiês de usuários. Em caso de incidente de segurança (LGPD Art. 48), notificaremos ANPD e titulares afetados no prazo legal, com base nos logs de borda (IP, Timestamp, Ação Bloqueada).

8. Seus Direitos como Titular (LGPD Art. 18) e Como Exercê-los LGPD Art. 18, 19, 20

Como não temos cadastro (login/senha) nem banco de dados relacional de usuários, o exercício de direitos funciona de forma autônoma (Self-Service) para dados locais e via requisição por e-mail para logs de segurança.

Confirmação e Acesso (Art. 18, I e II)

Dados no seu Navegador (Imediato, Autônomo): Abra as Ferramentas de Desenvolvedor (F12) > Aba Application > Local Storage > https://aipersonas.com.br. Lá estão: apiKeys, chatHistory, settings. Você vê, copia, exporta (JSON).

Logs de Acesso/Segurança (Cloudflare/GitHub): Não temos portal de autoatendimento. Envie e-mail para franckjlfp@gmail.com com assunto [LGPD ART LGPD ART 18 ] Solicitação Acesso Logs. Inclua: IP aproximado, período, prova de titularidade (ex: print do localStorage com sua chave API mascarada). Responderemos em 15 dias (Art. 18, §1º).

Correção (Art. 18, III)

Chaves API / Configurações: Edite diretamente na tela de Configurações do App (salva no localStorage instantaneamente).

Dados em Logs de Segurança (IP, User-Agent): São dados de observação técnica, não passíveis de "correção" subjetiva. Se houver erro factual (ex: geolocalização errada do Cloudflare), reporte ao provedor respectivo.

Eliminação / Exclusão (Art. 18, IV e VI)

⚡ Ação Imediata (Totalmente sob seu controle):

  1. No App: Botão "Limpar Todos os Dados Locais" (ou DevTools > Application > LocalStorage > Clear). Apaga Chaves, Histórico, Configs.
  2. No Provedor da IA: Revoque / Delete a Chave API usada (OpenRouter > Keys > Delete; Inception > API Keys > Revoke; Google AI Studio > API Keys > Delete). Isso invalida o segredo permanentemente.
  3. No Navegador: Limpe "Cookies e dados do site" para o domínio (remove cf_clearance, etc.).

Logs de Segurança (Cloudflare/GitHub): Não podemos apagar seletivamente seu IP de logs de firewall/acesso (são agregados, anonimizados, essenciais para segurança - Base Legal: Legítimo Interesse / Obrigação Legal). Eles expiram automaticamente (24h a 90d).

Conversas nos Provedores de IA: Você deve solicitar exclusão diretamente a OpenRouter, Inception Labs e Google (políticas deles). Nós não temos acesso a esses logs.

Portabilidade (Art. 18, V)

Seus dados estruturados (localStorage) já estão em formato aberto (JSON). Basta copiar do DevTools > Application > LocalStorage > "Exportar" (extensão) ou copiar/colar valores.

Não há formato proprietário. Não há "lock-in".

Oposição / Revogação Consentimento (Art. 18, §2º)

Como a base principal para logs é Legítimo Interesse, você pode se opor. Envie e-mail para franckjlfp@gmail.com. Avaliaremos se o interesse legítimo (segurança) prevalece (Art. 10, §2º). Para futuros cookies de analytics (Consentimento), haverá banner Opt-out fácil.

Decisões Automatizadas / Perfilamento (Art. 20)

Não realizamos decisões automatizadas que produzam efeitos jurídicos ou afetem significativamente seus interesses (ex: scoring de crédito, contratação automática, moderação automática de conta com banimento).

O WAF do Cloudflare bloqueia requisições maliciosas automaticamente (segurança), mas não "perfila" você como usuário do App para fins comerciais.

9. Proteção de Crianças e Adolescentes LGPD Art. 14; ECA Art. 17; Marco Civil Art. 20

Classificação Indicativa: 18+ (Dezoito Anos)

O AI Personas NÃO é dirigido a menores de 18 anos. As personas simulam "relacionamentos sintéticos", conversas íntimas, emocionais e de roleplay adulto. O conteúdo gerado por IA pode ser sexualmente sugestivo, violento ou psicologicamente complexo.

  • Não coletamos intencionalmente dados de menores de 18 anos.
  • Não há verificação de idade técnica (age-gating) na versão atual — responsabilidade do responsável legal.
  • Se você é pai/mãe/responsável e descobriu que um menor de 18 anos usa o App: faça-o parar imediatamente, limpe o localStorage do navegador e revogue a chave API. Contate-nos (franckjlfp@gmail.com) para orientação sobre logs de acesso.
